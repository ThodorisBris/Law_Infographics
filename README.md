# Law_Infographics
Automated Law Infographics using n8n and LLMs
Ένα καινοτόμο εργαλείο που στοχεύει στην ανάπτυξη μιας μεθοδολογίας για την αυτόματη παραγωγή Law Infographics που συνοψίζουν βασικά σημεία ενός νόμου μέσω της χρήσης LLMs. 

## Λειτουργία
Η διαδικασία χωρίζεται σε τρία βασικά στάδια:
1. **Διεπαφή Χρήστη:** Μια σελίδα HTML όπου ο χρήστης εισάγει σε φυσική γλώσσα το νομικό κείμενο.
2. **Αυτοματισμός (n8n):** Ένα workflow που λαμβάνει το κείμενο μέσω Webhook, επικοινωνεί με LLM και εξάγει βασικές έννοιες και λογικές σχέσεις.
3. **Οπτικοποίηση (Mermaid.js):** Μετατροπή της νομικής λογικής σε κώδικα διαγράμματος και παραγωγή εικόνας μέσω της υπηρεσίας Kroki.

## Τεχνολογικό Υπόβαθρο
* **n8n:** Ανάπτυξη της αυτοματοποιημένης ροής εργασιών.
* **LLMs (Gemini 2.5-flash):** Ανάλυση κειμένου και παραγωγή κώδικα Mermaid.
* **Mermaid.js:** Γλώσσα περιγραφής διαγραμμάτων.
* **HTML/JavaScript:** Για την αποστολή των αιτημάτων (HTTP Requests).
* **Kroki API:** Υπηρεσία rendering των διαγραμμάτων

## Οδηγίες Χρήσης
1. Κατεβάστε το αρχείο `Law Infographic n8n workflow.json` και εισάγετέ το (import) στο δικό σας **n8n**.
2. Ρυθμίστε τα API Keys για το γλωσσικό μοντέλο της επιλογής σας στον αντίστοιχο κόμβο.
3. Ανοίξτε το αρχείο `Webhook trigger` με την εφαρμογή Notepad και αντικαταστήστε το url με αυτό που εμφανίζεται στον αντίστοιχο κόμβο Webhook trigger στο δικό σας **n8n**.
4. Ανοίξτε το αρχείο `index.html` στον browser σας.
5. Επικολλήστε το νομικό κείμενο και πατήστε "Δημιουργία Διαγράμματος".
